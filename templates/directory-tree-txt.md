### Purpose
Directory tree visualization for projects.  
Helps students present file structures clearly.

### Example — Light Style
```
project
├── src
│   ├── index.js
│   └── utils.js
├── tests
│   └── test_main.js
└── README.md
```

### Variation — Heavy Style
```
project
┣━━ src
┃   ┣━━ index.js
┃   ┗━━ utils.js
┣━━ tests
┃   ┗━━ test_main.js
┗━━ README.md
```
