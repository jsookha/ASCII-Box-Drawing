### Purpose
Shows how generic classes, templates, and type parameters are represented in ASCII UML.

### Example — Generic Repository

```
┌──────────────────────────────────────────┐
│        Repository<T>                     │
├──────────────────────────────────────────┤
│ +items: List<T>                          │
├──────────────────────────────────────────┤
│ +add(item: T): void                      │
│ +getById(id: int): T                     │
└──────────────────────────────────────────┘
```

### Example — Generic Pair

```
┌──────────────────────────────────────────┐
│              Pair<K, V>                  │
├──────────────────────────────────────────┤
│ +key: K                                  │
│ +value: V                                │
├──────────────────────────────────────────┤
│ +swap(): Pair<V, K>                      │
└──────────────────────────────────────────┘
```

### Example — Generic Service With Constraints

```
┌──────────────────────────────────────────────────────────┐
│         DataService<T extends Entity>                    │
├──────────────────────────────────────────────────────────┤
│ +cache: Map<int, T>                                      │
├──────────────────────────────────────────────────────────┤
│ +save(entity: T): void                                   │
│ +find(id: int): T                                        │
└──────────────────────────────────────────────────────────┘
```
