### Purpose
Shows class inheritance relationships.  
Useful for OOP teaching or documenting class hierarchies.

### Example — Simple Tree
```
Animal
 ├── Mammal
 │    ├── Dog
 │    └── Cat
 └── Bird
      └── Eagle
```

### Example — UI Component Tree
```
Component
 ├── Button
 ├── Input
 └── Dropdown
      ├── SingleSelect
      └── MultiSelect
```
