### **Purpose**
Shows class inheritance relationships.  
Useful for OOP teaching or documenting class hierarchies.

### **Example 1 — Simple Tree**
```
Animal
 ├── Mammal
 │    ├── Dog
 │    └── Cat
 └── Bird
      └── Eagle
```

### **Example 2 — UI Component Tree**
```
Component
 ├── Button
 ├── Input
 └── Dropdown
      ├── SingleSelect
      └── MultiSelect
```
