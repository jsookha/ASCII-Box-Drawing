### Purpose
Shows common network layouts: star, mesh, ring, bus.

### Example 1 — Star Topology
```
             Switch
      ┌────┬────┬────┬────┐
      │    │    │    │    │
    Node Node Node Node Node
```

### Example — Mesh Topology
```
A ───── B
│ X │ X │
C ───── D
```

### Example — Ring Topology
```
A ── B ── C ── D ── A
```

### Example — Bus Topology
```
Node ── Node ── Node ── Node
```
