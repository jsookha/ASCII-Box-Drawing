### Purpose
Shows chronological sequences, milestones, and project phases.

### Example — Horizontal Timeline
```
2024 Q1 ────●───────2024 Q2──────●──────2024 Q3──────●──────2024 Q4
             Planning             Build               Test           Launch
```

### Example — Milestone Roadmap
```
Start
  │
  ├──● Requirement Gathering
  │
  ├──● Design
  │
  ├──● Development
  │
  ├──● QA & Testing
  │
  └──● Release
```

### Example — Multi‑Track Roadmap
```
Frontend:   ●───UI───●────Components────●────Release
Backend:    ●──API───●────Services──────●────Release
DevOps:     ●──Infra─●────CI/CD─────────●────Monitoring
```
