### Purpose
Shows Kubernetes cluster components in a simplified ASCII layout.  
Useful for teaching pods, nodes, deployments, and services.

### Example — Basic Cluster Overview
```
┌──────────────────────────────────────────────┐
│               Kubernetes Cluster             │
├──────────────────────────────────────────────┤
│  ┌───────────────┐   ┌───────────────┐       │
│  │   Node 1      │   │   Node 2      │       │
│  │ ┌───────────┐ │   │ ┌───────────┐ │       │
│  │ │   Pod A   │ │   │ │   Pod C   │ │       │
│  │ └───────────┘ │   │ └───────────┘ │       │
│  │ ┌───────────┐ │   │ ┌───────────┐ │       │
│  │ │   Pod B   │ │   │ │   Pod D   │ │       │
│  │ └───────────┘ │   │ └───────────┘ │       │
│  └───────────────┘   └───────────────┘       │
└──────────────────────────────────────────────┘
```
