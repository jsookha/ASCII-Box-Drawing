### Purpose
A box‑framed course list for teaching schedules or assignment outlines.

### Example 1 — Double Line Style
```
╔═══════════════════════════════╗
║       COURSE LIST             ║
╠═══════════════════════════════╣
║ • Programming Logic           ║
║ • Web Development Basics      ║
║ • Data Structures             ║
║ • Mobile App Scripting        ║
╚═══════════════════════════════╝
```

### Example 2 — Rounded Style
```
╭───────────────────────────────╮
│       COURSE LIST             │
├───────────────────────────────┤
│ • Programming Logic           │
│ • Web Development Basics      │
│ • Data Structures             │
│ • Mobile App Scripting        │
╰───────────────────────────────╯
```
