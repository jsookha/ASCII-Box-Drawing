### Purpose
A simple menu box for CLI programs or teaching exercises.  
Shows students how to frame options clearly.

### Example 1 — Light Style
```
┌───────────────────────────────┐
│           MAIN MENU           │
├───────────────────────────────┤
│ 1. Start Program              │
│ 2. Settings                   │
│ 3. Help                       │
│ 4. Exit                       │
└───────────────────────────────┘
```
