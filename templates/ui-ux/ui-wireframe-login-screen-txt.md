### Purpose
Provides a simple ASCII mockup of a login screen.  
Useful for early UX planning, teaching layout concepts, or documenting flows.

### Example — Basic Login Screen
```
┌──────────────────────────────┐
│          Login Page          │
├──────────────────────────────┤
│  Username: [______________]  │
│  Password: [______________]  │
│                              │
│        ( Login Button )      │
└──────────────────────────────┘
```
