### Purpose
Provides reusable ASCII UI components for mockups and documentation.

### Buttons
```
[  OK  ]
[ Cancel ]
[ Submit ]
```

### Inputs
```
[________________________]
[  Enter name here       ]
```

### Cards
```
┌──────────────────────────┐
│   Card Title             │
├──────────────────────────┤
│  Content goes here...    │
└──────────────────────────┘
```

### Modals
```
┌──────────────────────────────────┐
│            Modal Title           │
├──────────────────────────────────┤
│  Are you sure you want to exit?  │
│                                  │
│   [ Yes ]      [ No ]            │
└──────────────────────────────────┘
```
